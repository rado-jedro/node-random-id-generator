# node-random-id-generator

### node-random-id-generator is a basic tool to generate an unique, random ID 


## Installation

`$ yarn add @rado-jedro/node-random-id-generator`

`$ npm install @rado-jedro/node-random-id-generator`

## How to use?
### Create index.js and paste the code below:

`const randomID = require('@rado-jedro/node-random-id-generator'); `

` console.log(randomID(10));`

### Open console and type:
`node index.js`
